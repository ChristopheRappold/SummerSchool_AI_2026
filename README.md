# SummerSchool_AI_2026
Jupyter notebooks for the session of "Escuela de Verano IA y Conocimiento Abierto del CSIC"

1. Binary classification tutorial: H -> ZZ -> 4 leptons
2. YOLO26 in Python: Detection, Segmentation, Evaluation, and Fine-tuning
3. Pix2Pix with PyTorch - GPU/CPU Tutorial

## Installation for the requirements

The install script can be used to make sure that the the packages in the requirements are installed. It assumes uv or conda to be already installed.
